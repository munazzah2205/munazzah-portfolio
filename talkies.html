TALKIES APP

Concept project · Feature ideation · User flow design

Purpose

Talkies is a conceptual movie ticketing app designed to simplify the booking experience by addressing common usability gaps in existing platforms. The focus is on prioritizing user needs through clear flows, intuitive interactions, and visually refined, friction-free design. This project also served as a hands-on exploration to strengthen my UX thinking, flow structuring, and interaction design skills.

Insights That Shaped the Experience

To ground this exploration in real user behavior, a short survey was conducted alongside qualitative observations. The goal was to validate assumptions and identify patterns around how users discover, evaluate, and book movie tickets.

UX Focus Areas

Booking flow clarity
Seat selection usability
Reduced cognitive load
Visual hierarchy in dense screens
Friction-free checkout
Typography-led layouts

Experience Blueprint

Early Experience Exploration

The experience was shaped by mapping the end-to-end flow and breaking it into clear, manageable moments. Early explorations helped reason through structure, screen relationships, and user decision points—allowing assumptions to be tested, navigation to be refined, and a strong foundation to be established before moving forward.

Design & Component System

Bringing It Together

With the core experience defined, the focus shifted to aligning structure, interaction, and visual language into a cohesive system. Each decision—from layout to component behavior—was shaped to support clarity, reduce friction, and create a seamless booking experience grounded in user needs.
